## 3. ADSL 接入网

### 3.1. ADSL 的基本原理

ADSL（非对称数字用户线）利用已有的电话线（双绞线）提供宽带上网服务。它之所以称为「非对称」，是因为下行带宽（从互联网到用户）远大于上行带宽（从用户到互联网），这符合大多数用户的使用习惯——下载数据远多于上传。ADSL 使用频分复用技术，在同一根电话线上同时传输电话语音和网络数据，两者互不干扰。

### 3.2. ADSL 的频谱分配

ADSL 将电话线的频谱分为三个频段。0 到 4 kHz 用于传统电话语音。25 kHz 到 138 kHz 用于上行数据传输。138 kHz 到 1.1 MHz（或更高）用于下行数据传输。这三个频段互不重叠，通过分离器（splitter）将语音信号和数据信号分开。ADSL 的下行速率通常为 1.5 Mbps 到 8 Mbps，上行速率为 640 kbps 到 1 Mbps。

### 3.3. ADSL 的接入方式

用户的 ADSL 调制解调器通过电话线连接到本地电话公司的中心局（CO）。在中心局，ADSL 接入多路复用器（DSLAM）将来自多个用户的数据信号分离出来，并将其连接到 ISP 的网络。DSLAM 是 ADSL 接入网的核心设备，负责在电话线和 ISP 网络之间进行信号转换和复用。

## 4. 光纤接入网（FTTH）

### 4.1. 无源光网络（PON）

无源光网络是 FTTH 中最常用的架构。PON 使用无源的光分路器将一根光纤分成多根光纤，连接到多个用户家庭。下行数据通过广播方式从光线路终端（OLT）发送到所有光网络单元（ONU），每个 ONU 根据标识提取属于自己的数据。上行数据使用时分多路复用，每个 ONU 在分配的时隙中发送数据。PON 的「无源」意味着分路器不需要供电，维护简单。

### 4.2. FTTH 的架构

FTTH 有几种不同的架构。光纤到户（FTTH）是光纤直接铺设到用户家中，提供最高的带宽。光纤到楼（FTTB）将光纤铺设到建筑物内，最后几百米使用铜线。光纤到路边（FTTC）将光纤铺设到路边的机柜，最后一段使用铜线。距离越短，铜线段的带宽损失越小，性能越好。

### 4.3. FTTH 的优势

FTTH 相比 ADSL 有巨大的优势。带宽方面，FTTH 可以提供数百 Mbps 甚至 Gbps 级别的接入速率，远超 ADSL 的 Mbps 级别。距离方面，光纤的衰减远小于铜线，信号可以在更长的距离上保持高质量。可靠性方面，光纤不受电磁干扰，不受天气影响，故障率更低。此外 FTTH 支持对称的上下行速率，适合视频会议、云存储等双向高带宽应用。

## 5. 接入网中使用的 PPP 和隧道

### 5.1. PPP 协议

PPP（点对点协议）是接入网中常用的链路层协议。它在用户和 ISP 之间建立点对点的连接，提供身份认证（PAP 或 CHAP）、链路配置和数据封装等功能。PPPoE（PPP over Ethernet）将 PPP 封装在以太网帧中，使得以太网环境也能使用 PPP 的认证功能。ADSL 接入通常使用 PPPoE 来建立连接和进行用户认证。

### 5.2. 隧道技术

隧道技术用于在接入网中封装和传输网络层数据包。PPP over ATM（PPPoA）和 PPP over Ethernet（PPPoE）都是常见的隧道技术。在 ADSL 网络中，用户的 PPP 会话数据被封装在 ATM 信元或以太网帧中，通过接入网传输到 ISP 的接入服务器。隧道技术使得用户的网络层数据可以在不同的链路层技术上透明传输。
