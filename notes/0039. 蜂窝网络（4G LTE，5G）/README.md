## 3. 4G LTE 蜂窝网络

### 3.1. 架构和部件

4G LTE 网络由两大部分组成。无线接入网（E-UTRAN）包含基站（eNodeB），负责无线信号的发送和接收，管理无线电资源，处理无线接入相关的协议。核心网（EPC，演进分组核心）负责用户认证、移动性管理、IP 地址分配和与互联网的连接。EPC 包含多个功能实体：MME（移动管理实体）处理信令和移动性管理，SGW（服务网关）处理数据转发，PGW（PDN 网关）连接到外部互联网。

### 3.2. LTE 协议栈

LTE 的无线接口协议栈分为用户平面和控制平面。用户平面负责数据传输，从上到下依次为 PDCP（分组数据汇聚协议，负责加密和头压缩）、RLC（无线链路控制，负责分段和重组）、MAC（媒体访问控制，负责调度和混合 ARQ）、物理层（负责调制、编码和无线传输）。控制平面在 PDCP 之上还有 RRC（无线资源控制）层，负责连接管理和配置。

### 3.3. LTE 无线电接入网

LTE 使用 OFDMA（正交频分多路复用接入）作为下行传输技术，SC-FDMA（单载波频分多路复用接入）作为上行传输技术。OFDMA 将信道分为多个窄的子载波，不同用户可以被分配不同的子载波集合，实现了灵活的频率资源调度。LTE 支持 MIMO（多输入多输出）天线技术，通过多个天线同时发送和接收数据来提高速率和可靠性。

### 3.4. 附加功能——网络连接和功率管理

LTE 网络为移动设备提供了 IP 连接，每个设备被分配一个 IP 地址，可以访问互联网。LTE 的功率管理机制包括不连续接收（DRX）——设备在没有数据传输时关闭接收器以节省电量，周期性醒来检查是否有下行数据。LTE 还支持载波聚合，将多个载波捆绑在一起以提高峰值速率。

## 4. 全球蜂窝网络——网络的网络

全球蜂窝网络是一个「网络的网络」。每个国家有多个运营商，每个运营商有自己的蜂窝网络。当用户出国漫游时，其归属运营商与被访运营商之间通过漫游协议相互连接。被访网络为漫游用户提供服务，计费信息通过国际漫游交换中心（如 GRX/IPX）传回归属运营商。全球蜂窝网络的这种互联架构使得用户可以在世界各地使用移动服务。

## 5. 5G 蜂窝网络

### 5.1. 5G 的目标

5G 有三大目标。增强移动宽带（eMBB）提供更高的峰值速率（目标 20 Gbps）和更大的容量。超可靠低延迟通信（URLLC）提供毫秒级的端到端延迟和 99.999% 的可靠性，支持自动驾驶、远程手术等关键任务应用。大规模机器类型通信（mMTC）支持每平方公里百万级的设备连接，满足物联网的需求。

### 5.2. 5G 的关键技术

5G 的关键技术包括：毫米波通信——使用 24 GHz 到 100 GHz 的高频段，带宽极大但传输距离短；大规模 MIMO——基站使用大量天线（64 到 256 根），通过波束赋形精确指向用户，提高信号质量和容量；网络切片——将物理网络划分为多个逻辑切片，每个切片针对特定应用优化；边缘计算——将计算资源部署在网络边缘，减少延迟。

### 5.3. 5G 的应用场景

5G 的应用场景远超传统手机通信。增强移动宽带支持 4K/8K 视频、VR/AR 等沉浸式体验。超可靠低延迟通信支持自动驾驶车辆之间的实时通信、工厂自动化控制。大规模物联网连接支持智慧城市中的传感器网络、智能农业中的环境监测。5G 还催生了新的应用模式，如云游戏、远程医疗等。
